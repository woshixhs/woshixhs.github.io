    问题：

    • 1: Mac 平台安装webpack(本地安装), webpack命令不能执行，需要设置PATH
     使用shell的临时环境变量，（./node-module/.bin/）


     

* 2: 在npm中使用Webpack

    我们在package.json中设置一个快捷方式，运行webpack。

    ```package.json 文件
    ...
    "scripts": {
        "build": "webpack"
      }
    ... 
    ``` 
    现在我们直接运行`npm run build`命令实现与上面相同的功能。
